# Projeto final da cadeira de computação física

## Análise sentimental de plantas através de um sensor de umidade

O MVP (Minimum Viable Product) do projeto é o seguinte:

- Sensor de umidade
- Tela LCD de 16x2
- Arduino Uno
- Buzzer
- Potenciômetro
